## Usage
   1. Install requirements with 
      > `pip install -r requirements.txt`
   2. Run with uvicorn
       > `uvicorn main:app --host 0.0.0.0 --ssl-keyfile <yourkeyfile> --ssl-certfile <yourcertfile>`
